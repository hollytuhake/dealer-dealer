About This App

Why This App

This app was created for small businesses who manufacture products that require lead times. The intended user is a sales manager who needs to track 3 related data sets: products they manufacture, dealers who stock their product, and the orders from those dealers. This app allows the user to view, create, update and delete those items. The data sets in the app are connected, so making a change in one table changes that item in related data sets. This app also solves the problem of data normalization by using drop downs to select instead of typed text.
